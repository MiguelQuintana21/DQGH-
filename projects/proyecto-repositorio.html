Repositorio de tesis y memorias de estadías - Universidad Tecnológica de Escuinapa

Aplicación web para la visualización de documentos intelectuales en tiempo real y sistema de administrador integrado.

Duración: 4 meses
Dispositivos: Celular y Computadora
Tipo: Aplicación Web + Backend
Ver detalles Proceso de desarrollo
DESCRIPCIÓN DEL PROYECTO

Repositorio de Trabajos Académicos

Este sistema surge como respuesta a la necesidad de gestionar eficientemente los valiosos trabajos académicos producidos por los estudiantes. Es por ello que este proyecto tiene el objetivo de gestionar tesis y memorias de estadías con el fin de difundir y facilitar el acceso a estos trabajos académicos, promoviendo la colaboración, el intercambio de conocimientos y la visibilidad de los estudiantes.
